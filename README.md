# Pinout

A reference website for electrical plug and socket types, built with Next.js, React, and Tailwind CSS.

## Prerequisites

- Node.js 18+
- npm, yarn, pnpm, or bun

## Getting Started

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site.

## Build

```bash
npm run build
npm start
```

## Tech Stack

- [Next.js](https://nextjs.org) 16
- [React](https://react.dev) 19
- [Tailwind CSS](https://tailwindcss.com) 4
- TypeScript
